TweetSeeker is an app to help you find those tweets that are a pain to find otherwise. If you know the screenname of the user and approximately when they said it, then we can help you quickly narrow down the tweets you have to look at.

TweetSeeker grabs the most recent 3200 tweets from a user, and then allows you to dynamically specify the date range that you're interested in.

{% if user %}

If you're ready to try it out, place the twitter ID of the user you're interested in exploring the search box above, and hit Seek.

{% else %}

Although TweetSeeker supports use by unauthenticated users, we highly recommend you login have a consistent experience. This is due to Twitter's rate limiting restrictions. Without logging in, the system might stop unexpectedly.
